Comprehensive Sales Data Analysis for Awesome Chocolates (MS Excel)

This project involved a thorough analysis of sales data for Awesome Chocolates using Excel. Key activities included calculating descriptive statistics, performing Exploratory Data Analysis (EDA) with conditional formatting, and creating detailed sales reports by country using formulas and Pivot Tables. The project identified top-performing products, detected anomalies, and conducted a best-in-class analysis. Additionally, profit analysis was performed by combining data from different tables, and dynamic reports were generated for country-level sales. The project concluded with strategic recommendations on product discontinuation. Various Excel features such as Tables, formulas, Pivot Tables, conditional formatting, and charts were extensively utilized.
